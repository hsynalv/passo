C Hesabi Akisi

                  
Bu islemler icin once run baslatilmis olmali. Tek butonla C hesabi kaydedilir ve transfer/finalize istegi gonderilir.

                  

                    

                      
                      
                    

                    

                      
                      
                    

                  


                  

                    
                    
                    
1 = Ana#1↔Tutucu#1, 2 = Ana#2↔Tutucu#2… C bu ciftin guncel tutucusundan alir. Tek ciftte genelde 1.

                  


                  

                    
                    
Kapaliysa sadece C hesabina transfer yapilir.
